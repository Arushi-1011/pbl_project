Smart Speech Therapy Assistant

Dataset Cleaning & Dysarthria Detection Model Implementation

Dataset Preprocessing Script

This script performs resampling (16 kHz), silence trimming, fixed-length normalization (2 seconds), and MFCC extraction for both dysarthric and non-dysarthric speech samples.

View clean_dataset.py on GitHub

Model Training Script

This script implements a speaker-independent dysarthria detection model using MFCC features and a Random Forest classifier. Evaluation includes accuracy, precision, recall, F1-score, and confusion matrix visualization.

View train_model.py on GitHub

Experimental Results

  • Total Samples: 2104
  • Non-Dysarthric: 535
  • Dysarthric: 1569
  • Speaker-Independent Accuracy: 97.19%
  • Evaluation Metrics: Accuracy, Precision, Recall, F1-score
