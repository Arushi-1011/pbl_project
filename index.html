Smart Speech Therapy Assistant

A Generative AI-Based Framework for Automated Assessment and Personalized Intervention

Hypothesis / Project Statement

Can a Generative AI-assisted system automatically assess speech patterns and generate personalized therapy exercises while maintaining a human-in-the-loop supervision model?

Problem Statement

Speech therapy is manual, time-intensive, and difficult to scale. There is limited accessibility and personalization in current therapy systems. Existing AI systems focus only on speech classification and do not provide adaptive therapy generation or continuous progress tracking.

Literature Review / Market Research

  • CNN/LSTM-based speech disorder detection
  • ASR-based phoneme scoring systems
  • Static therapy mobile applications

However, no integrated Generative AI system exists for personalized therapy planning with human supervision.

Research Gap / Innovation

  • Baseline acoustic assessment model (Implemented)
  • MFCC-based feature extraction
  • Generative AI for therapy plan generation
  • Adaptive feedback system
  • Human-in-the-loop validation

System Methodology

Dataset

  • Dataset: TORGO (Female speakers F01, F03, F04)
  • Total files collected: 537
  • Valid processed samples: 535
  • Preprocessing: 16kHz resampling, silence trimming, fixed 2-second duration, corrupted file removal

Current Implemented Pipeline

Speech Input → Preprocessing → MFCC Feature Extraction → Random Forest Classifier → Acoustic Assessment Output

Future Extension

Assessment Output → Generative AI Module → Personalized Therapy Plan → Adaptive Feedback → Progress Tracking

Results & Analysis

Baseline Model Accuracy: 90.65%

Evaluation Metrics: Precision, Recall, F1-score across three speaker classes. This validates the effectiveness of preprocessing and acoustic feature extraction.

Ethics & Safety

  • No autonomous medical diagnosis
  • Human therapist approval required
  • Voice data privacy protection
  • Bias monitoring across speakers
