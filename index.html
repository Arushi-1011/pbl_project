Smart Speech Therapy Assistant

This project focuses on developing a smart speech therapy assistant that analyzes speech patterns and generates personalized therapy suggestions using artificial intelligence.

Problem Statement

Speech therapy requires continuous monitoring and personalized guidance, but access to trained professionals is limited and sessions can be costly. Existing digital tools mainly focus on detecting speech patterns without providing meaningful therapy recommendations. This creates a gap between speech assessment and effective intervention. A smarter and more adaptive system is needed to support speech improvement in a scalable and accessible way.

Literature Review

Existing research includes CNN/LSTM-based speech disorder detection, ASR-based phoneme scoring systems, and static therapy applications. However, most systems focus only on classification and do not integrate therapy personalization.

Research Gap

This project combines acoustic speech assessment with AI-driven therapy suggestion within a unified framework, bridging the gap between speech detection and personalized intervention.

System Methodology

Dataset

TORGO Dataset (Female speakers F01, F03, F04). Total files: 537. Valid processed samples: 535. Preprocessing included 16kHz resampling, silence trimming, and duration normalization.

Model

MFCC feature extraction followed by Random Forest classification. Baseline acoustic assessment validates speech pattern differentiation.
