Smart Speech Therapy Assistant

This project focuses on developing a smart speech therapy assistant that detects dysarthria using acoustic analysis and aims to provide AI-driven personalized therapy support in future phases.

Dataset / Input

• Dataset: TORGO Speech Dataset
• Non-Dysarthric Speakers: F01, F03, F04
• Dysarthric Speakers: FC01, FC02, FC03
• Total Samples: 2104
• Non-Dysarthric: 535
• Dysarthric: 1569
• Resampled to 16 kHz
• Silence trimmed
• Duration normalized to 2 seconds
• Feature extraction: MFCC (13 coefficients)

Model / Architecture

  • Feature Extraction: MFCC used for acoustic representation.
  • Model: Random Forest classifier.
  • Training Strategy: Speaker-independent split to prevent speaker leakage.
  • Evaluation: Accuracy, Precision, Recall, F1-score, Confusion Matrix.
  • Final Accuracy: 97.19%

Results & Analysis

Accuracy
97.19%

The model achieved 97.19% accuracy under speaker-independent evaluation. This confirms effective discrimination between dysarthric and non-dysarthric speech while maintaining generalization across unseen speakers.
